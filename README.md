# ecart

React + Vite shop: product grid from [DummyJSON products API](https://dummyjson.com/products), product details modal, and a functional cart (quantities and subtotal stay in sync).

## Scripts

```bash
npm install
npm run dev    # development server
npm run build  # production build
```

Built with [Vite](https://vite.dev/) and [React](https://react.dev/).
